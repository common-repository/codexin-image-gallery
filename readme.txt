Create a responsive image gallery with just a few clicks. Using this plugin you can create an unlimited number of galleries with minimum settings.

== Description ==
This is a simple photo gallery plugin that will help you to create professional image galleries with the lightbox feature within minutes.

= Features =
1. Simple and fast image gallery with a good user interface
1. Three prebuilt gallery layout to choose from - Grid style, Masonry style and Justified style
1. You can create an unlimited number of galleries
1. Flexible number of columns
1. Stunning lightbox feature using the famous [Photoswipe Javascript Gallery](https://photoswipe.com/) by Dmitry Semenov
1. Image caption and description can be shown or hidden in the lightbox for each image
1. SEO friendly structure
1. Image Hover Icons by [Flaticon](https://www.flaticon.com/)

= How this plugin works =
1. Create a gallery from WordPress dashboard
1. Copy the shortcode for the respective image gallery and paste inside your page or post

== Installation ==
= Install using WordPress dashboard =
1. Go to the WordPress dashboard.
2. Click on Plugins -> Add New
3. Type "Photo Gallery by Codexin" in the search box and hit enter.
4. Once you found our plugin, click on the "Install Now" button
5. Once installed, then click on the "Activate" button

= Install using FTP =

1. Download the plugin from WordPress plugin repository
2. Unzip the archive on your computer
3. Upload codexin-image-gallery directory to the /wp-content/plugins/ directory
4. Activate the plugin through the "Plugins" menu in WordPress
